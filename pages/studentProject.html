Project Overview

The Student Recommendation Web App was developed as a final-year project to explore how data-driven insights can assist students in selecting courses that align with their strengths and academic history. Many students face difficulty identifying subjects in which they are most likely to excel, often leading to less-than-optimal academic outcomes.

This prototype leverages machine learning models—including linear regression and association rule mining (Apriori)—to analyze historical enrollment and grade data from a mock database, predict future GPA, and recommend suitable courses. Built with Django, Python, and PostgreSQL, the system features an intuitive web interface with dynamic dashboards and input forms, demonstrating how predictive analytics can enhance academic planning and student success when applied to real-world data.
